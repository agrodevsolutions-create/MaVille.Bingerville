# Contribuer au projet MaVille

Merci de votre intérêt pour MaVille. Voici les règles et étapes simples pour contribuer.

1. Forkez le dépôt et créez une branche dédiée : `feature/ma-fonction` ou `fix/ma-correction`.
2. Ouvrez une Pull Request (PR) vers `main` avec une description claire.
3. Respectez le format minimal : HTML propre, classes utilitaires, et textes en français.
4. Tests visuels : ajoutez captures d'écran pour les modifications UI.
5. Ajoutez un patch minimal et descriptif dans le titre de la PR.

Checklist pour la PR :
- [ ] Description claire et screenshots si nécessaire
- [ ] Aucun secret dans le code
- [ ] Respect des règles d'accessibilité (contraste ≥ 4.5:1)

Si vous avez besoin d'aide, ouvrez une issue.

Merci !
