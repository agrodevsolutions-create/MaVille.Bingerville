# Politique de confidentialité (résumé)

Ce document décrit les principes de base pour la collecte, le stockage et la suppression des données utilisateurs et des preuves (photos, géolocalisation).

Principes :
- Minimisation : collecter uniquement les données nécessaires au signalement.
- Anonymisation : par défaut, les signalements peuvent être anonymes. Les données personnelles ne sont conservées que si l'utilisateur s'identifie volontairement.
- Rétention : les photos et données de signalement sont conservées pendant 1 an par défaut, sauf demande de suppression ou exigence légale différente.
- Accès : l'accès administratif aux preuves est restreint et journalisé.
- Suppression : procédure de suppression sur requête (contact via issue ou email indiqué dans `CONTRIBUTING.md`).

Considérations légales : conformité avec l'Ordonnance 2024-368 et les bonnes pratiques internationales. Pour une politique complète, adaptez ce document selon les conseils juridiques locaux.
